本文记录一些ES6语法学习中的一些笔记知识点，学习资料参考《阮一峰 ECMAScript 6 入门》.

简介&由来

ES6（全称ECMAScript），是JS的一个标准，于2015年6月正式发布，下面是一个简单的时间表
ECMAScript1.0 1997年
ECMAScript2.0 1998年
ECMAScript3.0 1999年12月
ECMAScript4.0 2008年7月（终止开发）
ECMAScript5.0 2009年12月
ECMAScript6.0 2015年6月

let & const

let，const 是ES6新增加的声明变量方法。(ES6共有六种方法，var function let const import class)

  • let于var的区别
    1.let声明的变量只在所在代码块有效
    2.let存在暂时性死区
    3.不存在变量提升
    4.不允许重复声明
    5.let 定义不为全局变量

  • const
    1.const定义的变量不可重新赋值，必须定义时初始化
    2.const定义属性是不可靠的
    3.声明的变量只在所在代码块有效

数组

  • 新增方法
    • Array.from() 将部署了iterator接口的数据结构转换成数组
    • Array.of() 初始化数组，对比new Array() 更加利于理解
    • Array.prototype.copyWithin(target[,start,end])
    • Array.prototype.find()&Array.prototype.findIndex()
    • Array.protptype.fill()
    • 三个遍历方法 entries(),keys(),values()
    • Array.prototype.includes()

Reflect&Proxy

这两个新的API经常放在一起配合使用，所以一起记录

下面是API

这两个接口，一个用来改变对象的默认方法，一个用来调用对象的默认方法。所以他们的defineProperty参数是完全相同的，共有以下13种，用法本文不具体展开讲解。
-apply(target, thisArg, args)
-construct(target, args)
-Reflect.get(target, name, receiver)
-set(target, name, value, receiver)
-defineProperty(target, name, desc)
-deleteProperty(target, name)
-has(target, name)
-ownKeys(target)
-isExtensible(target)
-preventExtensions(target)
-getOwnPropertyDescriptor(target, name)
-getPrototypeOf(target)
-setPrototypeOf(target, prototype)

tips:

  • proxy代理对象后重写方法会覆盖原方法，而不是继承。

    这样显然不是我们想要的效果，所以ES6提供了Reflect接口，用来调用对象原始的默认方法

  • proxy代理对象后操作实例化proxy对象会改变代理对象的属性值,但是他们不等
